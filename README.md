# Ягодка 🍒

MVP мобильного интернет-магазина свежих ягод, собранных на Камчатке.

## Стек

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

Мобильный формат: контент ограничен шириной 390px и центрирован на десктопе.

## Экраны

- `/` — Главная: баннер, лента «Наши ягоды», кнопка поиска магазина
- `/catalog` — Каталог: сетка 2×N, все 14 ягод
- `/product/[slug]` — Карточка товара: выбор веса (250 г / 500 г / 1 кг), описание
- `/cart` — Корзина
- `/profile` — Профиль

Нижний таббар: Главная · Каталог · Корзина · Профиль.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Данные

Товары захардкожены в [`lib/products.ts`](lib/products.ts). Изображения — в `public/images/`.
